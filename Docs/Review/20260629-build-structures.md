## 🔧 백엔드 렌즈 핵심 입장 (Claude)
- 고정 슬롯 + 기능 구조물 제약은 자유배치보다 안전(성능/그리핑 한정). 단 **패시브 공짜 가치**(공짜 XP·자동 힐·자동 사격)는 키팅-서바이벌 압박을 갉아먹음 → 구조물은 *압박 하 결정*을 만들어야.
- 아키텍처 = 기존 **미션 프레임워크 + 카드 v2** 패턴 1:1 재사용: `AFPSRBuildSlot` 앵커 + `AFPSRStructure` 폴리모픽 서브클래스 + `UFPSRStructureDataAsset` + 서버권위 인덱스선택. 신규 = **설치 RPC 하나**. 슬롯 점유 = GameState 복제.
- 포탑 = **경량 비-GAS** 필수(GAS=플레이어/보스만) + `ApplyDamage` 브릿지 재사용. 진짜 리스크 = 적500 타겟팅 비용 → significance/그리드/캡/throttle.
- **공짜 XP 생성기 = 최대 위험**(킬-위험-보상 분리 → 레벨/프리즈 cadence·난이도곡선 붕괴). 구조물 = 런당 임시. 게이팅 = 기존 카드/언락 추첨 재사용.

## 🎮 클라이언트 렌즈 핵심 입장 (Codex)
- 동의: 고정슬롯 안전·가독성↑, 공짜 가치(특히 XP생성) 위험, 포탑 비-GAS+ApplyDamage.
- 보완: "적이 부숨"만이 긴장 아님(파괴 위협=수리/방어 피로·캠핑화 위험) → **충전/시간제/킬연동/오버차지**가 더 나은 긴장 축. 역할 **고정 금지**(전담 빌더=장르 위반, 누구나 1~2초 전술결정).
- 구조물 = "캠핑 거점"이 아니라 **전투 동선에 잠깐 생기는 전술 증폭기**. 건설 타이밍 = **미션 보상 우선**. UX = 카드(무엇)+월드 고스트(어디).
- 포탑 = 자동 사냥꾼 X, **플레이어 핑/위치로 유도되는 교차사격**(DPS<플레이어, 연출 한 단계 낮게, "Turret Assist" 별도 표기).

## 토론 로그 요약
- **R1**: 백엔드가 4쟁점(장르/페이싱·아키텍처 재사용·성능+GAS·메타/이코노미) 입장 제시 → Codex가 대부분 수렴 + 3추가(건설=미션보상 / 긴장축 다양화 / 포탑=핑유도 교차사격) + 역할고정 경고.
- **R2**: 남은 트레이드오프 2개(포탑 파괴가능성·차단 여부) 수렴 → 둘 다 **비파괴·비차단** 합의. 차단형 "막는다"는 별도 바리케이드 1종으로 격리(U7 후). MVP 3종 확정.

## ✅ 합의 권고
1. **고정 슬롯**(자유배치 X) = 디자이너 배치 `AFPSRBuildSlot` 앵커(MissionSpawnPoint 패턴 재사용). 슬롯 흩어짐 + 좋은 슬롯=동선 리스크.
2. **비파괴 + 비차단 = 기본값** ⭐ — 구조물이 적 타겟셋·플로우필드 장애물에 안 들어감 → **적500 타겟팅/어그로/끼임 비용 0 + 플로우필드 동적장애물·U7 의존 제거**(가장 큰 성능/아키텍처 이득). 긴장 = 충전/탄약/시간제/오버차지(파괴 위협 아님).
3. **구조물 = 런당 임시**(프래그먼트/카드처럼 런 종료 소멸), 일시적 파워 스파이크(영구 캠핑 거점 X).
4. **역할 고정 금지** — 누구나 1~2초 전술결정(슬롯 접근→1키→3후보 즉시선택→1~2초 홀드 설치→즉시 복귀).
5. **건설 2단계**: ① **부여** = 카드추첨 재사용(신규 `EFPSROfferType::StructureGrant`, WeaponUnlock 동형·서버권위 인덱스선택, **미션 보상 우선**) → 구조물 토큰. ② **설치** = 월드 슬롯 짧은홀드 → **서버 RPC**(슬롯 빔+토큰 검증) → `AFPSRStructure` 스폰(복제). 유일 신규 넷코드.
6. **포탑** = 비-GAS 경량 + `ApplyDamage`, DPS<플레이어, 핑/위치 우선타겟 유도, 연출 한 단계 낮게, "Turret Assist" 별도 표기. 긴장 = 탄약/열 게이지·오버차지·20~40s 효율창.
7. **XP = 생성 금지, 증폭/정제만**(싸워야 가치 발생 → 공짜 경제 차단).
8. **MVP 3종**(전부 비파괴/비차단/충전·시간제): **① 센트리 포탑**(탄약/열, 오버차지) **② 메디컬 비콘**(시간제 장판+제한 충전, 무한 힐 X, 다운 보조) **③ XP 리파이너**(주변 발생 XP 증폭/정제만, 자체 생성 X). **우선 = 포탑+메디컬 비콘으로 MVP 닫고 XP 리파이너 3번째**(밸런스 리스크).
9. **UX** = 카드 UI(무엇을)+월드 슬롯 고스트/프리뷰(어디에). 슬롯 비주얼=바닥패드/천장라이트/번호/홀로프레임(원거리 가독)+미니맵/컴퍼스 마커. 점유 슬롯=아이콘/상태/충전링.
10. **아키텍처** = 미션 프레임워크 + 카드 v2 재사용, **폴리모픽 구조물 효과 서브클래스**(새 구조물=서브클래스1+DataAsset1, 중앙 0), 슬롯 점유=GameState 복제. IsDataValid/GetDescription 자동화 차용.

## ⚖️ 미해결 쟁점
- **바리케이드(차단형 1종)** = 사용자 원안의 "막는다" 판타지. 양 렌즈 합의 = **MVP 제외**, U7(플로우필드 높이/동적장애물) 이후 **"1종 한정"으로 평가**(전용 `BarricadeSlot`만·partial choke만·전체 통로 차단 금지·동적우회 비용 부담 시 "벽" 대신 **감속장/밀도 완충장**으로 시작). → 실질 미해결이라기보다 "연기 후 재평가" 합의.
- (경미) XP 리파이너 MVP 2종 vs 3종 — 밸런스 리스크로 3번째 권고.

## 🙋 사용자 결정 필요
1. **시스템 채택 여부** — 거점/유틸 레이어를 게임 정체성에 더하는 **기둥 결정**(양 렌즈: 비파괴/비차단/임시면 코어와 양립 판단, 그러나 채택 자체는 사용자 몫).
2. **타이밍/우선순위** — 전용 유닛, **코어 완전 잠금(U1 ✅) 후**. ⭐ 비차단 합의로 **U7 의존이 사라져** U7을 기다릴 필요 없음(바리케이드 옵션만 U7 후).
3. **건설 토큰 이코노미** — 미션보상 단독 부여 vs 레벨업 혼합(빈도/페이싱).
4. **XP 리파이너 포함** — MVP 2종(포탑+메디컬) vs 3종.
5. **바리케이드(차단형) 추구 여부** — 원안 "막는다"를 별도 1종으로 살릴지 / 순수 기능형으로 종결할지.
