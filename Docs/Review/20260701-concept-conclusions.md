## 🔧 백엔드 렌즈 핵심 입장 (Claude)
- 정보 비대칭·방향 위협 큐·FF 배율·데이터드리븐 스폰룰은 **시스템상 싸게** 이미 가능(U5 경고 생산자·FF 경로 재사용). 리스크는 시스템이 아니라 **클라 체감**(큐 폭주, 트롤 마찰, 조준 유의미성)에 있음.
- USP 재정의("잡몹=개인 손맛 / 협동=스페셜·사각·부활")는 서버 예산도 줄이고(HUD는 희소 위협만 브로드캐스트) 판타지도 명확하게 함.
- FF 비치명·난이도별·고의감쇠는 서버권위로 저비용 구현 가능 → 사용자 "FF ON"과 공개매칭 안전의 절충 가능.

## 🎮 클라이언트 렌즈 핵심 입장 (Codex)
- **잡몹은 오디오/압력으로 집계, 특수·원거리·DBNO·아군위험만 HUD로 선명하게.** 잡몹 화살표 수십 개 = 협동이 아니라 HUD 소음 = 실패.
- **FF 치명 50% 기본값은 공개매칭 최대 리스크**(트롤·사고사·아군 실루엣 가림). 기본은 **전면 비치명 FF** 권장, 치명 50%는 Hazard급 난이도 옵션으로.
- 스페셜 적은 원칙1 안에서 **타겟팅 룰+스폰+연출**로 가능(개별 AI 불요). MVP=기존 원거리 재색칠 1종.
- 1인칭 가독성은 **수치화 가능** — 계측 싼 5지표부터.

## 토론 로그 요약
- **R1**: 4쟁점(협동 성립/FF/스페셜 적/게이트 측정) 제시. Codex가 (1)(3)(5)(6) 방향 확인, (2)FF ON을 공개매칭 최대 리스크로 지목, 게이트 지표 대량 제시. "잡몹 집계/특수 선명" 원칙 도출.
- **R2**: (a) USP 재정의 합의 + Codex가 **초반 4솔로 공백**(스페셜 없는 0~4분) 신규 리스크 지목·값싼 fix 제시. (b) FF는 **비치명 전면 default** 최종 추천, "관통/AOE만 치명"은 기각(빌드 위축). (c) 게이트 지표를 계측 난이도별 3분류 → 1인 팀 최소 5지표 확정.

## ✅ 합의 권고
1. **USP 문장 확정** = *"스웜은 각자 갈아내고(개인 손맛), 팀은 서로의 사각·원거리·다운을 커버한다(협동)."* 잡몹은 협동 대상이 아니라 시야를 압박하는 **배경장**. → `Concept.md §1-C-3` 정련.
2. **HUD 원칙**: 잡몹 개별 UI 금지. 시각 위협 인디케이터 = **동시 ≤3 + 후방 집계 1**, 45도 섹터 병합, 밝기/펄스로 개수 표현. 오디오 경고 **동시 1·쿨다운 1.0~1.5s**. 우선순위 `즉시성<1.5s > 특수/핀/원거리 > DBNO/아군위험 > 근접잡몹`. → `PlayerFeel.md §2-14`.
3. **초반 4솔로 공백 방지**(신규): 0~60s 개인 손맛, 60s+ 양방향 스폰 압력, **75~180s 내 첫 원거리/스페셜-lite 1회**, 5분 전까지 DBNO·사각·원거리 최소 1회씩 경험. 팀 위협 비트 하한 = 1분 후 30~45s마다, 5분 후 15~25s마다 1회. → `RunFlow.md §2-8`·`Enemy.md §2-6`.
4. **협동유도 스페셜 적 = 원칙1 준수 가능**: `IsolationScore`(최근접 아군거리+시야밖+저최근구조) 타겟팅 룰 + 고립 플레이어 측후방 스폰 가중 + 연출(고유 사운드/색/HUD 마커). 개별 AI/StateTree 불요. MVP = 기존 원거리 베이스 재색칠 1종. 후보: Leasher(둔화/끌림)·Pinger(고립자 차징)·Splitter(파티 사이 스폰)·Screamer(aggro 편향). → `Enemy.md §2-6`.
5. **가독성 USP 게이트 최소 5지표**(1인 팀, 계측 싼 것): ① 활성 적 200~300(스트레스 500) ② 클라 relevant 적 P90 ≤150 ③ 화면 내 적 P50 ≤40 / P90 ≤70 ④ 15m 즉시 위협 P90 ≤25 ⑤ 시각 위협 큐 동시 ≤3(+후방집계1, 45도 병합). **이 5개 미통과 시 aim uplift·오탐율 등 상위 측정으로 가지 말 것.** 5개 통과=USP "가능성" 통과(확정 아님 — 10~15분 플레이에서 "4솔로 아니라 커버했다" 반복 확인 별도). → `Performance.md §5`.
6. 가성비 순위(협동 유도, 1인 팀): **스페셜 적 > 어그로 룰/카드 > FF→회복 카드**. FF→회복 카드는 오버힐 없음·틱 상한·다운직전 세이브 용도로만(지속 힐건化 금지).
