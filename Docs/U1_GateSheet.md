## 0. 세팅

- **빌드**: Development (에디터 PIE 또는 Standalone). 성능 측정은 **Standalone/패키지 권장**(에디터 오버헤드 분리). 멀티(V2·호스트 부하)는 **리슨서버 2-client**.
- **맵**: `L_Sandbox`(코어 검증 단일 맵).
- **2-client(V2·멀티 성능)**: PIE Net Mode = `Play As Listen Server`, Players = 2 (또는 별도 인스턴스 2개 + 로비 트래블).
- **디버그 표시**: `FPSR.RunDebug 1`(런 클럭/목표치/생존수 온스크린), `stat unit`·`stat fps`(프레임 분해), `stat game`(게임스레드), `stat net`(복제).

### 콘솔 명령 치트시트 (전부 `#if !UE_BUILD_SHIPPING`/`ECVF_Cheat`)
| 명령 | 용도 |
|---|---|
| `FPSR.EnemyTarget 500` | **디렉터 목표 생존수 설정**(0~하드캡 500). 점진 충원 — 성능 램프 측정용 핵심 노브 |
| `FPSR.SpawnEnemies 200` | 플레이어 주변 링으로 **즉시 버스트** N마리(스파이크 측정) |
| `FPSR.KillAllEnemies` | 전 적 회수(리셋) |
| `FPSR.RunTimeScale 4` | 런 클럭 가속(미션/보스타임 빨리 도달) |
| `FPSR.SkipToBoss` | 보스 페이즈 즉시 진입(보스+스웜 동시 부하) |
| `FPSR.SetFriendlyFire 1` | **V2 FF 토글**(1=on, 0=off) |
| `FPSR.AddXP 1000` / `FPSR.DrawCards` | 레벨업 프리즈·카드 추첨 강제(②·카드 흐름) |
| `FPSR.MissionTrigger` / `FPSR.MissionClear` | 미션 출현/클리어(프리즈 보상 흐름) |
| `FPSR.Pause` | 전역 프리즈 토글(②) |
| `FPSR.TestBlindspot` / `FPSR.TestDamageDir` | 사각 오디오·피격 방향(③) |

## B. §5 성능 측정 (정량 — Unreal Insights + NetProfiler)

### 측정 시나리오
1. Standalone 또는 2-client 리슨서버, `L_Sandbox`.

## C. V2 — 2-client 정합 (체크리스트 분리 = 실패 원인 격리)

### FF (Friendly Fire)
- [ ] `FPSR.SetFriendlyFire 1` → 아군 사격 = **50% 데미지** (⚠️ Roadmap §7-3의 10%는 stale, 실제 50%)
- [ ] FF OFF(`0`) = 아군 **무피해**(관통도 아군 통과·미정지)
- [ ] 바주카/유탄 = 아군 **런칭**(넉백은 데미지와 독립 — FF off여도 런칭)

### ChargeLaser (서버권위)
- [ ] 클릭 1회 → 자동 차징(warm-up 틱 소뎀 연속) → 완료 시 본뎀 1발
- [ ] 차징 중 조준 이동 = 빔 추적 / 차징 중 재클릭 = 무시(중복 차징 없음)
- [ ] 무기 교체 시 시퀀스 취소·무크래시
- [ ] 적·아군(FF)·관통 각각 데미지 판정 정상

### 🔵 W1 3차 신규 교정분 재확인(이번 머지 `b2c55d3`)
- [ ] **설정 오버레이 발사 래치(P2-2)**: 풀오토 발사 중 Esc로 설정 열 때 **무기가 멈추는가**(연사 지속 X)
- [ ] **적 500 net-freq(P2-1)**: 위 Part B 측정 중 NetProfiler에서 net-frequency 과호출/스파이크 없음(LOD 정상)

> **알려진 한계 = 버그 보고 금지**: Finding A(원격 클라 코스메틱 반동/게이트 레이스 — 데미지 무영향, U13 해소), 서버측 Hitscan 블룸=0-for-remote(2차 보류, U13/V2 netcode), 플레이어 넉백 오너클라 스무딩(후속).
